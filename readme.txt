Select columns and entries of your forms then export table in .xls. Requires Caldera Forms.

== Description ==

This is the long description.  No limit, and you can use Markdown (as well as in the following sections).

Features:
1. Unselect columns in Fields list tab
2. Select entries in Data table tab else all entries are selected
3. Expott in .xls

= Translations =

The plugin is translated into French and Spanish, but Data table language depending on your locale language and [Datatables i18n](https://github.com/DataTables/Plugins/tree/master/i18n)
You can [translate Contact Form 7](https://contactform7.com/translating-contact-form-7/) on [translate.wordpress.org](https://translate.wordpress.org/projects/wp-plugins/contact-form-7).

== Installation ==

1. Upload `twx-excel` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. This plugin need Caldera Forms installed
